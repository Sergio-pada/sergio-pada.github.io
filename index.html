PROJECTS

THE KEENE CAFE

This website was built for the historic Keene Cafe located at 30256 Woodford-Tehachapi Rd, Keene, CA. It features the restaurant's menu, a Google Maps view of the cafe's location, and a brief history of the cafe.

The site was built using HTML and CSS and is hosted through GitHub Pages.

STUDENT ROSTER

A program for sorting students by different attributes, such as their degree program and the average amount of time that they take to complete a course. The program is also capable of identifying incorrectly formatted student data.

This program was built in C++ and parses through an array of strings containing comma-delimited student data to create a list of student objects that are then sorted.

APPOINTMENT SCHEDULING APP

This desktop application is used to keep track of a company's appointments associated with a variety of customers and employees. Additionally, it is designed to handle appointment tracking across multiple countries, generate appointment schedules, and sort appointments by type and month.

The application was built using Java, JavaFX, and a MySQL Server. It implements a user authentication system and handles users across different time zones and countries through time zone conversion.

INVENTORY SYSTEM APP

The Product and Parts Management System is a desktop application designed to streamline and organize a company's data related to products and their constituent parts. This application serves as a comprehensive solution for managing inventory, pricing, and procurement details for both products and individual parts.

This project was built using Java for the logic and JavaFX for the user interface.

THE PALACE OF GREEN PORCELAIN

This website is an online archive of exhibits for a fictional museum based on the artifacts described in The Time Machine by H.G. Wells.

The site was built with HTML, CSS, JavaScript (Node.js), TensorFlow.JS, Chart.js, and PostgreSQL. It uses a machine learning model built with TensorFlow.js to recommend exhibits based on user activity and features data visualizations on user activity accessible via the "Dashboard" page.

PACKAGE DELIVERY SYSTEM

This Python simulation optimizes delivery routes across three package delivery trucks, minimizing the time delivery drivers spend on the road. The program facilitates real-time delivery tracking throughout the day and calculates the distance of each package to its destination based on the current time of day.

The project was built with Python, and packages are sorted on each truck using the nearest neighbor algorithm. It utilizes a custom hash table for the storage and retrieval of package object data.
