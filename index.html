PROJECTS

THE KEENE CAFE

This website was built for the historic Keene Cafe located at 30256 Woodford-Tehachapi Rd, Keene, CA. It features the restaurants menu, a Google Maps view of the cafe's location, and a brief history of the cafe.

This site was built using HTML and CSS. The site is hosted through GitHub Pages.

STUDENT ROSTER

A program for sorting students by different attributes, such as their degree program and the average amount of time that they take to complete a course. The program is also capable of identifying incorrectly formatted student data.

This program was built in C++ and parses through an array of strings containing comma-delimited student data in order to create a list of student objects that are then sorted.

COFFEE SHOP DATABASE REPORT

This SQL script sets up a relational database schema for managing information related to a fictional chain of coffee shops and their employees, suppliers, and coffee products. It also inserts sample data and creates a view for easy access to employee details. The creation of an index enhances the efficiency of queries involving the coffee_name column in the Coffee table.

APPOINTMENT SCHEDULING APP

This desktop application is used to keep track of a company's appointments associated with a variety of customers and employees of the organization. Additionally, it is designed to handle appointment tracking across multiple countries. The application can also generate appointment schedules for customers and sort the appointments by their type and month.

This application was built using Java, JavaFX, and uses a MySQL Server. The program implements a user authentication system. In addition, it defines and manipulates data related to customers, appointments, and users. It can handle users across different time zones and countries through time zone conversion.

INVENTORY SYSTEM APP

The Product and Parts Management System is a desktop application designed to streamline and organize a company's data related to products and their constituent parts. This application serves as a comprehensive solution for managing inventory, pricing, and procurement details for both products and individual parts.

This projust was built using Java for the logic and JavaFX for the user-interface.

THE PALACE OF GREEN PORCELAIN

This website is an online archive of exhibits for a fictional museum based on the artifacts described in The Time Machine by H.G. Wells.

This site was built with the use of: HTML, CSS, Javascript (Node.js), TensorFlow.JS, Chart.js, and PostgreSQL. The site uses a machine learning model built using TensorFlow.js to recommend exhibits to users based on their previous activity on the site. In addition to the machine learning model, the data on exhibits and users is aggregated and used for data visualizations on user activity accessable via the "Dashboard" page.

PACKAGE DELIVERY SYSTEM

This Python simulation is designed to optimize delivery routes across three package delivery trucks, minimizing the time delivery drivers spend on the road. The program not only facilitates real-time delivery tracking throughout the day but also calculates the distance of each package to its destination based on the current time of day.

This project was built with the use of Python. Packages are sorted on each truck using the nearest neighbor algorithm on each packages address. It utilizes a custom hash table for the storage and retreival of package object data.
